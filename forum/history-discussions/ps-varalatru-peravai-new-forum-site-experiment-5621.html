All,

You may remember that I created poll couple of months back after
hearing from Sivaram about creating forum site and the idea was
dropped. I was also skeptical at that time about it. Now I came across
the software phpbb3 which seems to be a good replacement for our yahoo
group. I imported all our messages to the forum and did some
experiments with various available options to see whether it can be
used for various purposes.

It is available at http://beta.ponniyinselvan.net

Earlier I was skeptical because the main difference between a
yahoo group site and a forum. The messages will reach you in yahoo
groups and in case of forums you have to go there to see the messages.
Since we are used to yahoo group site, forum site may not appeal. But
after checking the features and importing the existing messages, I
think it can be done. We can still maintain the yahoo group site, all
messages posted to yahoo groups can be archived here, same time if we
post any new messages or replies to existing topics in the forum, then
an email can be triggered to yahoo group site which everybody will
get. In this case, both yahoo group and the forum will be in sync and
both can be used. After a while if we think that the forum site is
very good, we can discontinue yahoo group, otherwise we can simply
maintain it as a archive site and continue with yahoo group.

It has lot of very useful features. It is available at
http://www.phpbb.com/about/features/. I have created few forums/topics
to demo the features. I have imported all the users as your yahoo id
and email id as your yahoo email. You should be able to login after
resetting the password through
http://beta.ponniyinselvan.net/ucp.php?mode=sendpassword.

1. Supports UTF-8 tamil fonts, you can see it in the home page
where I have used both english and tamil fonts.
2. All the existing messages are available in this forum - 20276
messages 5172 topics.
http://beta.ponniyinselvan.net/history-discussion-f12/
3. We have 638 members who contributed at least one messages out
of 1410+ members, you can see the member list and their post here -
http://beta.ponniyinselvan.net/memberlist.php, You need to login to
look at the list.
4. Easy to moderate, remove offensive topic, changing the topic
title, splitting topic, stick it to the top, easy to edit at any point
of time if the information is incorrect (I believe we can't do this
right now with yahoo group).
5. Different forums for different type of discussions
6. Seperate forum for Articles eg., -
http://beta.ponniyinselvan.net/novel-characters-f21/vallavarayan-vanthiy...
7. Attachments in Download forum -
http://beta.ponniyinselvan.net/downloads-f14/
8. Links Forum - http://beta.ponniyinselvan.net/links-f16/
9. Fancy stuff - embedding youtub, google videos -
http://beta.ponniyinselvan.net/history-discussion-f12/pallava-chola-temp...,
http://beta.ponniyinselvan.net/history-discussion-f12/vaaram-oru-alayam-...
10. Sticking important message at the top - (TAMIL HERITAGE
FOUNDATION :: Meeting is on top)
http://beta.ponniyinselvan.net/history-discussion-f12/
11. Search features
12. List of topics, posts written by members
12. We can create more forums to facilitate our new initatives and
discussions which can be private and available to only few members.

Lot more, it is always based on what we need, it is extensible.

Still this is an experiment, based on the usage, maintenance,
server load either I will abandon this or will discontinue the current
archive site and maintain this.

Please let me know your inputs, since we are doing lot of things
recently, we can make this better and present it as a good source for
others who are seeking it.

Regards
Thiru
